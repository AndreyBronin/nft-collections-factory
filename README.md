# How to launch demo

1. Install deps
```
npm install
```

2. Run tests(optional)
```
npm run test
```

3. Run hardhat node
```
npm run node
```

4. Deploy contract
```
npm run deploy
```

5. Send 1 ETH to testing address
```
npx hardhat --network localhost faucet 0xaaabbbccc
```

6. Run server
```
npm run server
```

7. Run frontend
```
cd frontend
npm i
npm start
```
